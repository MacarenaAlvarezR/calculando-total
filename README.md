# 🧮 Calculando Total - Product Card

Proyecto web interactivo que simula el cálculo del precio total de un producto según la cantidad seleccionada.

Desarrollado con **HTML, CSS y JavaScript puro**, enfocándose en la manipulación del DOM y eventos.

Pages link : https://macarenaalvarezr.github.io/calculando-total/
---

## ✨ Funcionalidades

- ➕ Aumentar cantidad de producto
- ➖ Disminuir cantidad de producto
- 💰 Cálculo automático del total
- 📦 Actualización dinámica en pantalla

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
